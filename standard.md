# RJ45 standard

## Pin Table
| pin | data    | Colour |
|-----|---------|--------|
| 1   | gnd     | w/g    |
| 2   | 5v      | g      |
| 3   | SDA     | w/o    |
| 4   | CAN-A H | b      |
| 5   | CAN-A L | w/b    |
| 6   | SCL     | o      |
| 7   | CAN-B H | w/br   |
| 8   | CAN-B L | br     |

# CAN Bus Standard
## Switches
- Switch for selecting 8/16MHz operating mode
- Switch for terminating CAN Bus
